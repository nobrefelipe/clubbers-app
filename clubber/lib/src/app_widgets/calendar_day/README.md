### **Calendar Day Widget**
This widget renders a calendar date.

We use it when we need to show a month/day combination.

### **Parameters**

`CLDatetime date`


### **How to use**
```dart
final date = CLDatetime('21-10-2021 20:18:04Z');

CalendarDayWidget(date)
```
